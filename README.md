Pharmacy Management System (Using Java)

A simple GUI for pharmacy customers. Part of Object Oriented Programming-1 [JAVA] course at AIUB.

**Features**
* Customers can register by providing required information
* Secure password and email validation for customer login
* Customers can select medicines from list and add to cart
* Checkout system with receipt generation
* Customers can log out or delete their account

**Characteristics**
* Basic project showcasing ability to create user-friendly GUI
* Single page GUI design
