Pharmacy Management System(Using Java)

It is a simple GUI(Graphical User Interface) for customers of a pharmacy. It is a part of Object Oriented Programming-1[JAVA] course of AIUB. 

Features:

*Customers can register them by providing required information
*There are safety of password and email id for customer log-in
*Customers can select their required medicine from medicine list and can add them to cart
*Customers can checkout from cart and will get a receipt
*Customers can log-out as well as delete their account

Characteristics:

*Basic project to showcase the capability of a student of creating a user friendly GUI
*Single page GUI
